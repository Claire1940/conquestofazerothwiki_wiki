<Section icon={Target} title="Class-Specific Tuning That Keeps Combat Readable">

The strongest Conquest of Azeroth ElvUI profiles are not the most packed ones. They are the ones that match your playstyle. Melee builds usually need more center-screen clarity. Ranged builds often benefit from tighter cast visibility. Pet-heavy or support-heavy kits need extra space for trackers and proc alerts.

| Playstyle | Main UI Stress | Best Adjustment | ElvUI Feature |
|---|---|---|---|
| Melee | Fast target swaps | Strong target frame anchor | Cast bar, nameplates |
| Ranged | Position checks | Cleaner center lane | Cast bars, resource text |
| Hybrid | Mixed priorities | Flexible bar grouping | Action bar paging |
| Support | Buff tracking | Aura filtering | Buff/debuff filters |

| Priority | What to Show | What to Hide |
|---|---|---|
| High | Interrupts, cooldowns, resources | Decorative borders |
| Medium | Long buffs, procs, utility bars | Rarely used macros |
| Low | Extra chat noise, duplicate trackers | Repeated frame elements |

<Callout type="warning" title="Avoid Over-Tuning">
If every class gets its own wildly different layout, you will spend more time managing profiles than playing. Keep a shared baseline and only adjust the essentials.
</Callout>

<Section icon={Shield} title="Troubleshooting, Backup Habits, and a Final Checklist">

If your Conquest of Azeroth ElvUI setup feels messy, the fix is usually structural, not cosmetic. Start by removing duplicate frames, lowering the number of visible bars, and checking whether a boss mod or combat addon is stacking information in the same space.

| Issue | Likely Cause | Fast Fix |
|---|---|---|
| Frames overlap | Too many elements centered | Move utility frames outward |
| Combat feels crowded | Bars too large | Reduce bar height and spacing |
| Important buffs vanish | Weak aura filters | Raise aura priority rules |
| Nameplates are noisy | Too many indicators | Trim nonessential markers |
| UI feels inconsistent | Mixed profiles | Rebuild from a single baseline |
