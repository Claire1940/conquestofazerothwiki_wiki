The progression structure is rigid but highly rewarding. As you level up from 1 to 60, you earn talent points that must be spent alternately [file:///root/Documents/GameProjects/0_meta/conquestofazerothwiki_wiki/00首页信息-2.md]. | Level Milestone | System Unlocked | Point Allocation Rule |
| **Level 1-9** | Base Class Kit | No talent points available |
| **Level 10** | Specialization Choice | Unlock Class and Spec Trees |
| **Level 11-60** | Alternating Points | Class Tree (even), Spec Tree (odd) |
| **Level 10+** | Auto-Build System | Automated point placement |

<Callout type="info" title="System Mechanics">
Points spent in the Class Tree unlock general utility, while points spent in the Specialization Tree unlock role-specific power. You cannot hoard points to dump them all into a single tree [file:///root/Documents/GameProjects/0_meta/conquestofazerothwiki_wiki/00首页信息-2.md].
