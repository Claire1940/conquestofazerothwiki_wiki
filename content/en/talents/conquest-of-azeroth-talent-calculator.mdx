<Section icon={Sparkles} title="How the Conquest of Azeroth Talent System Works">

To truly master your character, utilizing the **conquest of azeroth talent calculator** is essential for mapping out your path. Conquest of Azeroth (CoA) features 21 completely original, Warcraft-inspired classes, each featuring unique spells, resources, and playstyles. To support this massive depth of character customization, the developers implemented a comprehensive talent overhaul heavily inspired by modern dual-talent tree designs. 

In older versions of the alpha, classes suffered from clunky pacing, excessively long tooltips, and passive spell bloat in the spellbook. The modern system solves this by splitting your talent progression into two distinct, parallel paths: the **Class Tree** and the **Specialization Tree**. The Class Tree houses generic utility, defensive, and baseline offensive tools shared by all members of your class. The Specialization Tree, unlocked at level 10, contains highly specific nodes that define your primary role, whether you choose to play as a tank, healer, DPS, or hybrid support.

As you level up from 10 to 60, you alternate between earning Class Tree points and Specialization Tree points. This alternating pacing forces meaningful decision-making at every level, ensuring that your character's power grows steadily without overwhelming your rotation.

| Level Bracket | Point Type Awarded | Tree Spent In | System Unlocked |
|---|---|---|---|
| **Level 1-9** | None | None | Base class kit training |
| **Level 10** | Specialization Point | Spec Tree | Primary specialization choice |
| **Level 11** | Class Point | Class Tree | Class-wide utility nodes |
| **Even Levels** | Class Point | Class Tree | Alternating progression path |
| **Odd Levels** | Spec Point | Spec Tree | Alternating progression path |
| **Level 60** | Capstone Point | Both Trees | Final tier capstone talents |

<Callout type="success" title="Alternating Point Strategy">
Because points alternate between Class and Specialization trees, plan your progression so that key rotation-enabling talents are prioritized on odd levels, while utility and defensive nodes are filled on even levels.
</Callout>
