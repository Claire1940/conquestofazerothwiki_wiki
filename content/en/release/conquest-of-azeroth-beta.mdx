To play on the newly launched Vol'Jin realm, players must transition from the old beta PTR client to the official live client. The installation process is handled entirely through the unified Project Ascension launcher, which manages game patches, client configurations, and approved user add-ons. 

| Requirement | Specification | Notes |
| :--- | :--- | :--- |
| **Operating System** | Windows 10/11 (Direct) / Linux (Community Guide) | Official launcher natively supports Windows systems. |
| **Account Type** | Single unified Ascension Account | One account grants access to all realms and forums. |
| **Launcher Version** | Latest Project Ascension Launcher | Required to download, patch, and launch the custom client. |
| **Disk Space** | Approximately 20-30 GB of free space | Solid State Drive (SSD) is highly recommended for loading times. |

<Steps>
  <Step num="1" title="Create an Ascension Account">

  </Step>
  <Step num="2" title="Download and Install the Launcher">
    Navigate to the official download page and download the Windows client launcher. Run the installer and choose your preferred directory.
  </Step>
  <Step num="3" title="Select the Live Realm Path">
    Open the launcher, log in with your credentials, and locate the realm selection dropdown menu. Ensure you select the live **Conquest of Azeroth** path rather than the PTR options used during the beta.
  </Step>
  <Step num="4" title="Patch and Launch the Game">
    Click the "Play" or "Install" button to let the launcher download the custom game files. Once the progress bar is complete, the launcher will automatically apply any required custom patches.
  </Step>
</Steps>
